    # Maintain the virtual freezer and lab journal

    The virtual freezer contains reagents, oligonucleotides, positive controls and qPCR programs which can be linked to assays and experiments. We highly encourage to keep the virtual freezer well maintained in order to ensure complete documentation. In all categories entries can be created or uploaded as batches.

    Download the entries as a excel file:

    • Select the appropriate entries and choose Download Excel in the drop down list.
    • Click Run Action button.

    The Lab Journal section manages the Standard Operating Procedures (SOP) which can be linked to the assay and experiments.
