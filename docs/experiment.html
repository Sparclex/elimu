# Create a experiment

The experiment links the Assay with the Samples and data is directly attached.

  • Click Create Experiment button.
  • Copy-paste the Sample IDs in to the Samples section. Use a new line for each Sample ID.
  • Select the appropriate Assay.
  • Save the experiment by clicking on the Create Experiment button.

Upload (qPCR) data:

  • Go to the Experiments overview
  • Edit the created Experiment by clicking on Edit (Notepad symbol).
  • Upload the qPCR data file by choosing the data file.
  • Save the experiment by clicking on the Update Experiment button.

# Review and download the results

  • Go to the Results section.
  • Use the Assay filter to select for the assay of choice.
  • By using the Result filter samples which need attention can be identified, e.g. select Errors.
  • Select all samples of interest. Click on Select action and download the excel file by clicking on the Play symbol.

Change the validation status (exclude/include) of the qPCR replicates:

  • Click on the View symbol of the sample.
  • Review the replicates in the Data section. Select the appropriate replicate(s) and click on Select action to change the validation status. Confirm by clicking on the Play symbol.
  • Change the validation status.
